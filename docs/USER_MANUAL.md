## üîß Troubleshooting

### App Won't Start

**Symptoms:** App crashes on launch

**Solutions:**
1. Force stop and restart app
2. Clear app cache (Settings ‚Üí Apps ‚Üí Doctor App ‚Üí Storage)
3. Reinstall app if issue persists
4. Check device storage space

### Data Not Saving

**Symptoms:** Changes don't appear after saving

**Solutions:**
1. Check internet connection (for cloud sync)
2. Restart the app
3. Verify sufficient device storage
4. Try again with simpler data
5. Restore from backup if corruption suspected

### Backup Failed

**Symptoms:** Backup process doesn't complete

**Solutions:**
1. Check available storage space
2. Check cloud account (if using cloud backup)
3. Try backup to local storage instead
4. Restart device and try again
5. Contact support if issue persists

### Search Not Working

**Symptoms:** Search results appear empty or incorrect

**Solutions:**
1. Check search keywords are correct
2. Ensure data exists in system
3. Clear search filters
4. Refresh app (pull down)
5. Check if data was properly saved

### Notifications Not Showing

**Symptoms:** Appointment reminders not appearing

**Solutions:**
1. Check notification settings enabled
2. Verify app has notification permissions
3. Check device notification settings
4. Disable and re-enable in app settings
5. Restart app

### Performance Issues (Slow Loading)

**Symptoms:** App feels sluggish, slow to load screens

**Solutions:**
1. Restart device
2. Clear app cache
3. Reduce number of loaded records (archive old ones)
4. Close other apps running in background
5. Check device storage (less than 500MB may affect performance)

### Sync Issues

**Symptoms:** Data not syncing between devices

**Solutions:**
1. Check both devices on same network
2. Ensure both apps are same version
3. Restart both apps
4. Use backup/restore to manually sync
5. Check cloud account status

### Storage Issues

**Symptoms:** App running out of space

**Solutions:**
1. Archive old records to reduce database size
2. Export and delete old invoices
3. Delete backup files you don't need
4. Clear app cache
5. Move app to SD card (if device supports)

### Printing Issues

**Symptoms:** Can't print prescriptions or invoices

**Solutions:**
1. Verify printer is connected and online
2. Check printer has paper
3. Ensure device on same WiFi as printer
4. Try printing to PDF first
5. Restart printer and device

### Import/Export Issues

**Symptoms:** Can't import backup or export fails

**Solutions:**
1. Verify file is correct format
2. Check file isn't corrupted
3. Ensure sufficient space for export
4. Try different export location
5. Contact support with file details
